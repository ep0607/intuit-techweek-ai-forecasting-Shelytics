# Shelytics – AI-Powered Lending Intelligence

## Overview

Shelytics is an AI-powered lending intelligence platform built during the Intuit AI/ML Hackathon at NYC Tech Week 2026.

Our solution helps small-business lenders make smarter funding decisions by predicting loan default risk, forecasting portfolio performance, and evaluating counterfactual lending scenarios. The goal is to maximize portfolio profitability while maintaining transparency and explainability in every lending decision.

---

## Problem Statement

Traditional lending decisions often rely on historical outcomes and static risk models that struggle to explain:

* Which applicants should be funded
* The uncertainty behind predictions
* How portfolio performance changes over time
* What would happen under alternative lending interventions

Lenders need reliable, interpretable, and data-driven tools to make profitable lending decisions.

---

## Solution

Shelytics combines machine learning and predictive analytics to:

* Predict loan default probability
* Estimate confidence intervals for risk predictions
* Forecast portfolio default trajectories
* Evaluate counterfactual lending outcomes
* Support explainable decision-making

Our platform empowers lenders to balance profitability and risk while understanding the reasoning behind each recommendation.

---

## Features

### Risk Prediction

Predicts the probability that a loan applicant will default.

### Portfolio Forecasting

Projects portfolio-level default behavior across future cohorts.

### Uncertainty Estimation

Provides confidence intervals around predictions to improve trust and decision-making.

### Counterfactual Analysis

Evaluates how outcomes would change under hypothetical interventions.

### Explainable Insights

Generates transparent reasoning behind funding recommendations.

---

## Dataset

The challenge dataset consisted of approximately 100,000 small-business loan applications collected over 18 months, including:

* Business identity information
* Financial metrics
* Credit utilization
* Banking activity
* Platform engagement
* Historical lending outcomes
